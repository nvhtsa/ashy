# Simple Version

Drop this folder's contents into the root of your GitHub Pages repo.

## Quick edits
- **Tab title:** `index.html` -> `<title>...</title>`
- **Favicon:** replace `assets/img/favicon.ico`
- **Logo:** replace `assets/img/logo.png`
- **Background video:** replace `assets/media/background.mp4`
- **Music:** replace `assets/media/music.mp3`
- **Links:** edit the `href` values in `index.html`
